# Advent of Code 2022, Python

## Requirements
- Python 3.10
- Poetry

## Install deps
```shell
poetry install
```

## Run tests
### All days
```shell
./tests.sh
```
### Specific day
```shell
./tests.sh day01
```

## Run a day
```shell
poetry run python -m day01
```
